Git

1.查看状态

git status

2.拉取远程

git pull

3.查看分支

git branch -a

白色文本：本地分支，红色文本：远程分支

4.切换分支

git checkout 分支名

5.添加代码

git add .

6.提交代码

git commit -m “提交信息”

7.创建本地分支

git checkout -b 分支名

8.删除分支

删除未切换的分支：git branch -d 分支名
删除已切换的分支：git branch -D 分支名

9.查看提交信息

git log

10.显示分支差异

  • 显示出分支A和分支B中差异部分（代码增加或减少的个数）：git diff 分支A 分支B –stat

  • 显示指定文件的详细差异（代码可视化）：git diff 分支A 分支B 具体文件路径

  • 显示出所有有差异的文件的详细差异（超级全）：git diff 分支A 分支B

11.计算文件MD5值

md5sum.exe 文件路径

12.更新子模块

git submodule update –init –recursive

若报错，先删除子模块文件夹
